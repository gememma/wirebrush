# wirebrush

[![ci](https://github.com/gememma/wirebrush/actions/workflows/ci.yml/badge.svg)](https://github.com/gememma/wirebrush/actions/workflows/ci.yml)

## Project

Project wirebrush is the name given to the personal website hosted at [gemmat.dev](https://gemmat.dev).
It is written in Rust using [actix-web](https://actix.rs/), [maud](https://maud.lambda.xyz/) and 
[pulldown-cmark](https://lib.rs/crates/pulldown-cmark).
